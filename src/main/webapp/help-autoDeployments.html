Enable this setting to send deployment data to Jira whenever any pipeline on this server completes a deployment stage with a name matching the regex filter specified.
Deployment stages must include an environment name (for example "deploy to prod") to send environment data to Jira. Use (?<envname>.*)$ in the regex filter to extract and send environment names.
Only use this setting if your teams have a consistent naming convention for deployment stages in their Jenkinsfiles.
